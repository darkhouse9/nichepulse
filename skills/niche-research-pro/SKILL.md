## Workflow

### Step 1: Run NichePulse

```bash
nichepulse list --top 20
nichepulse score urban-beekeeping
nichepulse compare "urban-beekeeping herbalism foraging"
nichepulse roadmap urban-beekeeping
nichepulse custom "My Custom Niche Idea"
```

### Step 2: Load Extended Database

Read the full niche database from `pro/data/niche-database.json` for detailed analysis. Each niche includes:
- Scores (5 dimensions + total + grade)
- Competition level and Etsy result counts
- Google Trends 12-month change
- Reddit and Facebook community sizes
- Average order value estimates
- Seasonal peak times
- Top-performing product types
- Gift and identity scores

### Step 3: Seasonal Timing Check

Before launching, check `pro/data/seasonal-calendar.json` to time the launch for peak buying season. Launching a hammock camping design in January = wasted opportunity. Launch in May-June = catch the wave.

### Step 4: Generate Roadmap

For niches scoring 16+, create a 90-day launch roadmap using the template from `pro/roadmaps/`. Each roadmap includes:
- Phase 1: Foundation (Days 1-14) — designs + shop setup
- Phase 2: Validation (Days 15-35) — reviews + community building
- Phase 3: Optimization (Days 36-60) — kill losers, double winners
- Phase 4: Scale (Days 61-90) — multi-channel + ads

### Step 5: Competitor Tracking

Use `pro/competitor-tracker.md` template to monitor:
- New entrants in the niche weekly
- Price changes from top sellers
- Design trends and gaps
- Review sentiment analysis

### Step 6: Validation Checks

Run these for any niche scoring 14+:

1. **Etsy Demand**: Search niche on Etsy. Top 5 listings with 100+ sales = validated
2. **Google Trends**: 12-month trend flat or up = green light
3. **Community**: Active subreddit/FB group with 5K+ members = audience exists
4. **Design Gap**: Can you spot 3+ gaps in top 20 Etsy results? = opportunity
5. **Uniqueness**: Can you imagine 20+ unique designs without repeating? = depth

**Score: 5/5 = START NOW | 4/5 = Strong | 3/5 = Test with 5 designs | ≤2 = Skip**
